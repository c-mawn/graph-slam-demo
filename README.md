# graph-slam-demo
Demo of Graph SLAM for Olin College Discrete Math Fa'25: Ivy Mahncke, Dom Salmieri, Charlie Mawn

## Setup

Clone the repository.

In the root directory of the repo, create a Python virtual environment with 

```bash
# Linux
python3 -m venv venv
```

```bash
# Windows
python -m venv venv
```

Activate the venv, then install requirements with

```bash
pip install -r requirements.txt
```

If using `pip-tools`, use `pip-compile` to generate `requirements.txt` and `pip-sync` to install dependencies.
